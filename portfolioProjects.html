Knight vs Foe

Work In Progress!

Written in C#, the thought behind this came as I was learning how objects work and how I can get two classes to interact with each others properties. By instantiating an object from each class I was able to simulate a battle between a player and randomly generated foes, creating new values and using a choice/accuracy algorithm each turn to make battles less predictable. At the same time it challenges the player to win by killing as many enemies in the least amount of turns possible.

Home-Page Animation


A homepage should be simple and welcoming, however not too bare. To create a landing page to entice guests I decided to make a simple animation using JavaScript to increase the letter spacing at a smooth rate. There are 3 points towards the end of the animation in which the expansion rate of the letter spacing slows down. Making it almost seemless with no sudden jumps or stops.

Colour Flash Animation


Generates a number of <div> elements with a function that assigns a random colour and changes the border radius to create an animation. It removes the previous div's on screen and uses 'setInterval' to replace them with a new batch assigned with new random colours, whilst following the value of the previous radius making them seem as if changing shape.



List Creator

Work In Progress!


This is a list creator which give user freedom in terms of adding, removing and moving the list around. It takes full advantage of the DOM tree which allows for these features to work. The event listeners check that the function you are selecting, then matches it by the elements Id; making relevant changes such as 'appending' new list items and then attaching the user-functional buttons to them. More functions to be added soon!
