Player vs Foe

Work In Progress!

Written in C#, the thought behind this came as I was learning how objects work and how I can get two classes to interact with each others properties. By instantiating an object from each class I was able to simulate a battle between a player and a randomly generated foe, creating new values and using a choice/accuracy algorithm each turn to make battles less predictable. At the same time it challenges the player to win in the least amount of turns possible.

Home-Page Animation


A homepage should be simple and welcoming, however not too bare. To create a landing page to entice guests I decided to make a simple animation using JavaScript to increase the letter spacing at a smooth rate. There are 3 points towards the end of the animation in which the expansion rate of the letter spacing slows down. Making it almost seemless with no sudden jumps or stops.
