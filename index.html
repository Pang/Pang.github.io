A full stack web developer profficient in .Net Core technologies and front-end Javascript frameworks, primarily Angular and Vue. I currently work as a Software Developer / Dev Ops Engineer for the NHS, where my main focus is on the Nexus Electronic Patient Record. This is a multi-purpose system designed to improve upon the hospitals admission flow, having a clean interface to understand the wards current state and which patient needs tending to, as well as connecting together the PAS and other independant systems into one place for medical and administration staff to access.

Second to this I am a Cofounder and developer of Copilot, this is a platform for Twitch data tools including but not limited to: profile design analytics, generated graphs to see how users interacted with your streams, music detection to avoid DMCA copyright strikes and more.

Apart from web-development, I am also passionate about producing music, drawing, snowboarding, and making fun things in Unity :)


Copilot

Copilot is a collection of data tools for Twitch as well as a place to collaborate and gain insights with other streamers. This is my latest project with another peer and allows for real-time graph generation of phrase highlighting on previously saved streams. Features for Copilot are plentiful and still continues to grow!

ThoughtWall

Thoughtwall was my first project to learn Angular and DotnetCore. It has since gone through many iterations to get to it's current state and uses technologies such as SignalR (Websockets/RPC's), Angular Material (Themes and components) and Entity Framework Core (ORM). This app allows users to register and create threadposts where conversations can take place in real time.
